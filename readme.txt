Given score of 3 dice, Choose the next best move.
That is, By re rolling which dice/ dices will serve a better score than the current score

Score= sum of the rolled numbers

On an Yahtzee, that is when all the three dice have same value, the score will be 25.

Note that a dice can have a value in range of 0-6.


Input should be give in this format:

python yahtzee.py 3 5 2
